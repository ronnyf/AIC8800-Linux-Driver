### Task 7: Fix rwnx_wakelock.c for kernel 6.x

**Fixes:** Deprecated wakeup_source_create/add API removed in kernel 6.x

**Files:**
- Modify: `drivers/aic8800/aic8800_fdrv/rwnx_wakelock.c`

**Steps:**
- [ ] **Step 7: Update rwnx_wakeup_init function**

At lines 12-18, replace:

```c
struct wakeup_source *rwnx_wakeup_init(const char *name)
{
	struct wakeup_source *ws;
	ws = wakeup_source_create(name);
	wakeup_source_add(ws);
	return ws;
}
```

With:

```c
struct wakeup_source *rwnx_wakeup_init(const char *name)
{
	struct wakeup_source *ws;

#if LINUX_VERSION_CODE >= KERNEL_VERSION(6, 0, 0)
	ws = wakeup_source_register(NULL, name);
#else
	ws = wakeup_source_create(name);
	wakeup_source_add(ws);
#endif
	return ws;
}
```

- [ ] **Step 8: Update rwnx_wakeup_deinit function**

At lines 20-26, replace:

```c
void rwnx_wakeup_deinit(struct wakeup_source *ws)
{
	if (ws && ws->active)
		__pm_relax(ws);
	wakeup_source_remove(ws);
	wakeup_source_destroy(ws);
}
```

With:

```c
void rwnx_wakeup_deinit(struct wakeup_source *ws)
{
	if (ws) {
		if (ws->active)
			__pm_relax(ws);
		wakeup_source_unregister(ws);
	}
}
```
